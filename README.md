# About
This project has a number of experimental Views.
## Views
### Meta (unfinished)
A view selector.
### Wrigger (unfinished)
Procedural model gen using self-organizing space-filling curves.
### Swarm (unfinished)
Compute shader based swarm/flocking sim.
### MeshViz (unfinished)
Mesh viewer with rudimentary STL support.
